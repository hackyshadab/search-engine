
# Getting started

## System requirements 

* It supports in all OS.


## Search engine Tool installation on unix

```
> apt-get update -y

> apt-get install python3 -y

> pip3 install Flask

> apt-get install git -y

> git clone https://github.com/hackyshadab/search-engine.git

> cd search-engine

> python3 app.py 

```
